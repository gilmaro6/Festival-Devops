🎵 Festival DevOps Music Fest

Proyecto desarrollado como evidencia del programa DevOps y Contenedores del SENA.
📌 Descripción

El proyecto consiste en una Landing Page para el evento Pacific DevOps Music Fest, desarrollada con HTML y CSS, acompañada de una API REST construida en Flask y desplegada mediante Docker y Docker Compose.

Durante el desarrollo se aplicó un flujo básico de trabajo con Git y GitHub utilizando ramas (Git Flow básico), commits y fusiones (merge).
🛠 Tecnologías utilizadas

    HTML5
    CSS3
    Python 3
    Flask
    Docker
    Docker Compose
    Git
    GitHub
